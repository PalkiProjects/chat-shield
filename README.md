# Chat Shield

A Real-Time Secure Chat Application developed using Flutter and Firebase.

---

## 📌 Project Description

Chat Shield is a modern real-time messaging application designed to provide fast, secure, and responsive communication between users. The application is developed using Flutter for frontend development and Firebase for backend services such as Authentication and Cloud Firestore Database.

The project demonstrates implementation of cloud-based real-time communication systems where users can create accounts, log in securely, and exchange messages instantly. Firebase Cloud Firestore enables real-time synchronization of messages without refreshing the application.

This project was developed as a B.Tech Computer Science & Engineering major project to understand mobile application development, Firebase integration, and real-time database management systems.

## ⚙️ Installation Steps

### Step 1 – Clone the Repository

```bash
git clone https://github.com/PalkiProjects/chat-shield.git
```

---

### Step 2 – Open the Project

Open the project in:

- Android Studio
- VS Code

---

### Step 3 – Install Dependencies

Run the following command:

```bash
flutter pub get
```

---

### Step 4 – Configure Firebase

1. Create a Firebase project.
2. Enable Firebase Authentication.
3. Enable Cloud Firestore Database.
4. Download the `google-services.json` file.
5. Paste the file inside:

```text
android/app/
```

---

### Step 5 – Run the Application

```bash
flutter run
```
